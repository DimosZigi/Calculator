# Calculator
A calculator made in Python with the use of Tkinter

What the prject does is creating a GUI with buttons. Each button coresponds to a number or an operational symbol:

(+) --> addition/n
(-) --> subtraction/n
(*) --> multiplication/n
(/) --> division
